## Причина
Ошибка `Invalid login credentials` от Supabase Auth означает одно из:
1. **Пользователь не зарегистрирован** - нет такого email в базе
2. **Неверный пароль** - пароль не совпадает с сохраненным
3. **Email не подтвержден** - но в этом случае была бы другая ошибка

## Рекомендации для пользователя

### Если вы видите ошибку "Invalid login credentials":

1. **Проверьте правильность учетных данных**:
   - Email должен быть в формате `name@kode.ru`
   - Email должен начинаться с буквы (не цифры)
   - Пароль должен совпадать с тем, что вы вводили при регистрации

2. **Если вы еще не регистрировались**:
   - Нажмите "Создать аккаунт"
   - Заполните все поля (имя, фамилия, email, пароль)
   - Подтвердите email с помощью кода из письма

3. **Если вы забыли пароль**:
   - К сожалению, функция восстановления пароля пока не реализована
   - Вам нужно обратиться к администратору или создать новый аккаунт

4. **Если вы зарегистрировались, но не подтвердили email**:
   - Нажмите "Создать аккаунт" с тем же email
   - Система отправит новый код подтверждения
   - Введите код из письма

## Canvas Grid - Будущая реализация

Canvas версия сетки календаря требует:
1. Правильную обработку sticky элементов с учетом scroll
2. Корректную работу событий (клики, hover) поверх canvas
3. Z-index стратегию для DOM элементов (события, header)
4. HiDPI support для четкого отображения на Retina
5. Оптимизацию перерисовки (только измененные области)

**Статус**: Отложено до решения базовых проблем с аутентификацией
