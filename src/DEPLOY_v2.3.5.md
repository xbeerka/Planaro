## 🔍 Диагностика проблем

### Проблема: "Остались пустые ячейки"
**Симптомы**: В консоли для некоторых сотрудников показывается `⚠️ Остались пустые ячейки: N шт.`

**Решение**:
1. Проверьте что используется новый алгоритм (должен быть 3-фазный)
2. Проверьте `maxIterations` - должно быть 1000
3. Если N < 10 ячеек - это нормально (edge cases)
4. Если N > 50 ячеек - откройте issue с логами

### Проблема: "Генерация медленная"
**Симптомы**: Генерация для 50 сотрудников занимает > 1 минуты

**Решение**:
1. Проверьте CPU usage браузера (Task Manager)
2. Попробуйте закрыть другие вкладки
3. Проверьте что задержка между сотрудниками 10ms (не 1000ms)

### Проблема: "Некоторые события не загрузились"
**Симптомы**: В консоли `⚠️ Создано N событий, но загружено только M`

**Решение**:
1. Проверьте что endpoint `/events` использует `.limit(50000)`
2. Проверьте Network tab - есть ли ошибки 500/503
3. Проверьте логи Edge Function в Supabase Dashboard
