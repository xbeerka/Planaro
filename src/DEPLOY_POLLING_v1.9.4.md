## 🎯 Как это работает

### События (уже было в v1.9.3)
```typescript
useEffect(() => {
  const pollEvents = async () => {
    // Защита от дублирования
    if (Date.now() - lastLocalChangeRef.current < 2000) return;
    
    // Запрос данных
    const data = await eventsApi.getAll(accessToken, workspaceId);
    
    // Обновление только при изменениях
    setEventsState(prev => {
      if (JSON.stringify(prev) !== JSON.stringify(data)) {
        // Обновить UI и кэш
        return data;
      }
      return prev;
    });
  };
  
  setInterval(pollEvents, 10000); // Каждые 10 секунд
}, [accessToken, workspaceId]);
```

### Сотрудники, Департаменты, Проекты (новое в v1.9.4)
Идентичная логика, только:
- Интервал: **15 секунд** (вместо 10)
- Ref: `lastResourcesChangeRef` / `lastDepartmentsChangeRef` / `lastProjectsChangeRef`
