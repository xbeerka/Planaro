# Быстрый тест исправления Undo v3.3.10

## 🎯 Что тестируем

Исправление race condition между Undo и debounced save при создании/удалении событий.

---

## ✅ Test Case 1: Undo сразу после создания события

### Шаги:
1. Открыть календарь
2. Выбрать любую ячейку
3. Создать событие (Ctrl+E или контекстное меню → Создать событие)
4. **Подождать 100ms** (событие создастся на сервере)
5. Сделать Undo (Ctrl+Z)
6. **Подождать 1 секунду** (debounced save должен был сработать)
7. Проверить консоль браузера

### Ожидаемый результат:
```
✅ 🔄 UNDO/REDO: Очистка pending операций для 1 удалённых событий...
✅    🧹 Очищена pending операция для: e37367
✅ 🔄 UNDO/REDO: Удалённые события успешно синхронизированы с сервером
✅ Нет ошибки "BATCH update: событие не найдено в БД"
```

### ❌ Старое поведение (БАГ):
```
❌ [Supabase] ❌ BATCH update: событие e37367 не найдено в БД
❌ BATCH update error: Event not found
```

## 🎨 Визуальная проверка

### Поведение событий:
1. ✅ Событие создаётся → спиннер → спиннер исчезает через ~500ms
2. ✅ Undo → событие исчезает мгновенно
3. ✅ Redo → событие появляется мгновенно
4. ✅ Нет "мигания" или "воскрешения" событий через 1-2 секунды

---

## 🐛 Если тест провален

### Симптомы:
- ❌ Ошибка "BATCH update: событие не найдено в БД"
- ❌ События "воскрешаются" после Undo
- ❌ Нет логов "Очистка pending операций"

### Что проверить:
1. `handleUndo()` вызывает `cancelPendingChange()` для каждого удалённого события
2. `handleRedo()` вызывает `cancelPendingChange()` для каждого удалённого события
3. `cancelPendingChange` есть в dependencies обоих useCallback
4. Pending операции очищаются ДО вызова `syncDeletedEventsToServer()`

## 📊 Benchmark

### Производительность:
- Создание события: **~500ms** (временный ID → реальный ID)
- Undo: **мгновенно** (восстановление из истории)
- Очистка pending: **< 1ms** (удаление из Map)
- Debounced save: **500ms** (но НЕ срабатывает для удалённых)

### Память:
- Pending операции очищаются сразу → **нет утечек памяти**
- Удалённые события помечаются в `deletedEventIdsRef` на 60 секунд
