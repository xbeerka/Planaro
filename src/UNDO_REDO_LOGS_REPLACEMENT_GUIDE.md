# Инструкции по замене логов Undo/Redo

## Цель
Добавить единый префикс `🔄 UNDO/REDO:` ко всем логам, связанным с Undo/Redo операциями, чтобы можно было легко отфильтровать их в консоли.

## Файлы для изменения

1. Найти: `console.log('↩️ Undo: МГНОВЕННОЕ восстановление из истории');`
   Заменить: `console.log('🔄 UNDO/REDO: ↩️ Undo начат - мгновенное восстановление из истории');`

## Фильтрация в консоли браузера

После изменений можно отфильтровать только логи Undo/Redo, введя в поиске консоли:
```
🔄 UNDO/REDO
```

Или в Chrome DevTools:
```
Filter: UNDO/REDO
```
