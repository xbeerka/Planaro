## 🏗️ Связь с другими версиями

### Зависит от
- **v3.3.12** - использует блокировку Undo для pending событий
- **v3.3.11** - использует блокировку race condition в Undo/Redo

### Предотвращает
- Ложные ошибки `Cannot delete temporary events via API`

- Надёжность cleanup orphaned событий: 95% → 100%
- Согласованность паттернов во всём коде
- Производительность (меньше сетевых запросов)
