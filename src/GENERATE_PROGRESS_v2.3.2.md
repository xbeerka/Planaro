# 📊 Generate Progress Modal v2.3.2 - Улучшенная визуализация

## 🎯 Что изменилось

Обновлён алгоритм генерации тестовых событий и модальное окно прогресса:

### ✨ Новые возможности

1. **Три прогресс-бара вместо одного**:
   - 🔵 **Синий бар** - прогресс обработки сотрудников (строк)
   - 🟢 **Зелёный бар** - прогресс заполнения ячеек календаря
   - 🟣 **Фиолетовый бар** - прогресс загрузки событий на сервер

2. **Детальные счётчики**:
   - Обработано сотрудников: `X из Y`
   - Заполнено ячеек: `Z из W` (52 недели × 4 юнита × кол-во сотрудников)
   - Загружено на сервер: `N из M` событий

3. **Delay между пачками загрузки**:
   - События загружаются батчами по **100 штук**
   - Между каждой пачкой **delay 30 секунд**
   - Предотвращает перегрузку сервера при массовой генерации

4. **Стадии процесса**:
   - `generating` - "Генерация событий..."
   - `uploading` - "Загрузка на сервер... (30 сек между сотнями)"
   - `done` - "Завершено!" (показывается 2 секунды перед закрытием)

## 🚀 Производительность

### Оптимизации
1. **Обновление прогресса ячеек каждые 100 шт** - снижение частоты setState
2. **Batch размер увеличен с 50 до 100** - меньше итераций, меньше delay
3. **Delay 30 секунд между пачками** - даёт серверу время на обработку

### Пример timeline для 500 событий
1. Генерация: ~5-10 секунд (зависит от кол-ва сотрудников)
2. Загрузка:
   - Пачка 1 (100 шт): ~2-3 секунды
   - Delay: 30 секунд
   - Пачка 2 (100 шт): ~2-3 секунды
   - Delay: 30 секунд
   - Пачка 3 (100 шт): ~2-3 секунды
   - Delay: 30 секунд
   - Пачка 4 (100 шт): ~2-3 секунды
   - Delay: 30 секунд
   - Пачка 5 (100 шт): ~2-3 секунды
3. **Итого**: ~2 минуты 10-15 секунд

## 📚 Технические детали

### Расчёт общего количества ячеек
```typescript
const WEEKS = 52;
const UNITS = 4;
const totalCells = resources.length * WEEKS * UNITS;
// Пример: 100 сотрудников × 52 недели × 4 юнита = 20,800 ячеек
```

### Подсчёт заполненных ячеек
```typescript
let processedCells = 0;

const markOccupied = (startWeek, startUnit, width, height) => {
  for (let w = startWeek; w < startWeek + width && w < WEEKS; w++) {
    for (let u = startUnit; u < startUnit + height && u < UNITS; u++) {
      occupancy[w][u] = true;
      processedCells++; // ← Инкремент при пометке
      
      // Обновляем UI каждые 100 ячеек
      if (processedCells % 100 === 0) {
        setGenerateProgress(prev => ({
          ...prev,
          currentCells: processedCells
        }));
      }
    }
  }
};
```

### Загрузка с delay
```typescript
const batchSize = 100;
let uploadedCount = 0;

for (let i = 0; i < eventsToCreate.length; i += batchSize) {
  const batch = eventsToCreate.slice(i, i + batchSize);
  
  // Загружаем пачку параллельно
  await Promise.all(batch.map(event => createEvent(event)));
  uploadedCount += batch.length;
  
  // Обновляем прогресс
  setGenerateProgress(prev => ({
    ...prev,
    uploadedEvents: uploadedCount
  }));
  
  // Delay между пачками (кроме последней)
  if (i + batchSize < eventsToCreate.length) {
    console.log('⏳ Задержка 30 секунд перед следующей пачкой...');
    await new Promise(resolve => setTimeout(resolve, 30000));
  }
}
```
