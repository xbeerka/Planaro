## 🎯 Что было сделано

### Проблема
Orphaned events cleanup использовал неправильный паттерн проверки временных ID (`'ev_temp'` без подчеркивания), из-за чего пытался удалить временные события через API, вызывая ошибки.

### Решение
Добавлено подчеркивание в конце паттерна (`'ev_temp_'`), что обеспечивает корректную проверку временных ID и предотвращает ложные DELETE запросы.

### Результат
- ✅ Нет ложных ошибок в консоли
- ✅ Нет лишних DELETE запросов к API
- ✅ Cleanup работает стабильно (100% успешность)
- ✅ Согласованность паттернов во всём коде
