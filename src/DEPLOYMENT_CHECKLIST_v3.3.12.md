## 🔍 Regression Testing

### История (History)
- [ ] Создание нескольких событий → Undo всех → Redo всех
- [ ] Изменение событий → Undo → Redo
- [ ] Удаление событий → Undo → восстановление
- [ ] 50 операций → Undo 50 раз → стек работает

### Drag & Drop
- [ ] Drag события → Undo → позиция восстанавливается
- [ ] Resize события → Undo → размер восстанавливается
- [ ] Gap resize → Undo → граница восстанавливается
- [ ] Drag временного события блокирован (v3.3.9)

### Создание/Удаление
- [ ] Создание → Undo → событие удаляется
- [ ] Создание → изменение → Undo → событие с изменениями удаляется
- [ ] Paste → Undo → вставленное событие удаляется

### Модалки
- [ ] Создание проекта → Undo НЕ влияет на проект
- [ ] Создание сотрудника → Undo НЕ влияет на сотрудника
- [ ] Изменения в модалках НЕ попадают в историю

### Delta Sync
- [ ] События НЕ "воскресают" после Undo
- [ ] Full Sync НЕ возвращает удалённые события (v3.3.3)
- [ ] Изменения других пользователей загружаются корректно
