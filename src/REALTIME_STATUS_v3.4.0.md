## 🔍 Что выяснилось

### ❌ Проблема

**Ошибка сборки**:
```
Error: Build failed with 1 error:
virtual-fs:file:///utils/supabase/client.ts:28:34: ERROR: [plugin: npm] Failed to fetch
```

**Причина**:
- Пакет `@supabase/supabase-js` **недоступен** в среде Figma Make
- Динамический импорт `import('@supabase/supabase-js')` не работает
- Ошибка происходит на этапе **сборки**, а не runtime

### ✅ Решение

**Создана заглушка в `/utils/supabase/client.ts`**:
```typescript
export async function getSupabaseClient() {
  console.warn('⚠️ @supabase/supabase-js недоступен в Figma Make');
  console.warn('⚠️ Supabase Realtime отключен - используйте HTTP polling');
  return null; // Всегда возвращает null
}

export async function isSupabaseRealtimeAvailable(): Promise<boolean> {
  return false; // Всегда возвращает false
}
```

**Результат**:
- ✅ Приложение собирается без ошибок
- ✅ Realtime Presence gracefully отключён
- ✅ Курсоры не отображаются (не критично)
- ✅ Приложение работает стабильно в режиме v3.3.6
