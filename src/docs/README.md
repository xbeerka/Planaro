## 🐛 Troubleshooting

### Проблема: "Мигание" всё ещё есть

**Решение**:
1. См. `/docs/TESTING_CACHE_CLEANUP.md` → раздел "Отладка проблем"
2. Проверьте логи браузера - есть ли `🧹 Очистка кэша`?
3. Проверьте IndexedDB - очищается ли `cache_online_users_batch`?

### Проблема: Аватарка не исчезает 60+ секунд

**Решение**:
1. См. `/docs/PRESENCE_LEAVE_FIX.md` → раздел "Troubleshooting"
2. Проверьте серверные логи - есть ли `👋 Leave`?
3. Проверьте Network tab - отправляется ли DELETE запрос?

### Проблема: Другие пользователи пропадают

**Решение**:
1. См. `/docs/TESTING_CACHE_CLEANUP.md` → "Сценарий 4"
2. Проверьте IndexedDB - должны остаться только ДРУГИЕ пользователи
3. Проверьте фильтр в App.tsx - `user.email !== currentUserEmail`
