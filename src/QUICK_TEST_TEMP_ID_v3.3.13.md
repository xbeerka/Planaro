# Быстрый тест исправления временных ID v3.3.13

## ⚡ Быстрая проверка (1 минута)

### 1. Создайте событие и сразу удалите департамент
```
1. Откройте календарь
2. Создайте новое событие (Ctrl+Click или правый клик)
3. НЕ ЖДИТЕ 500ms - сразу откройте модалку департаментов
4. Удалите департамент, к которому привязано событие
5. Сохраните изменения
```

**Ожидаемое поведение**:
- ✅ В консоли НЕТ ошибок `❌ Ошибка удаления события ev_temp_XXX`
- ✅ Orphaned event НЕ удаляется через API (пропускается)
- ✅ Cleanup выполняется через 5 секунд без ошибок

**Было до v3.3.13**:
- ❌ В консоли: `❌ Ошибка удаления события ev_temp_1732005123456_789`
- ❌ В Network: DELETE запрос с ошибкой 400
- ❌ Сервер: `Cannot delete temporary events via API`

## 🧪 Полное тестирование (5 минут)

### Тест 1: Создание → Cleanup → Без ошибок
```
1. Создайте 5 событий подряд (быстро!)
2. Откройте DevTools → Console
3. Удалите департамент со всеми событиями
4. Подождите 5 секунд (cleanup таймер)
5. Проверьте логи
```

**Результат**:
- ✅ Все временные события пропущены
- ✅ Только события с реальными ID удаляются через API
- ✅ Нет ошибок 400 в Network

### Тест 2: Race condition с createEvent
```
1. Создайте событие
2. СРАЗУ (через 100ms) удалите ресурс
3. Дождитесь создания события на сервере (~500ms)
4. Проверьте что событие получило реальный ID
5. Cleanup через 5 секунд должен удалить событие БЕЗ ошибок
```

**Результат**:
- ✅ Временное событие не удаляется через API
- ✅ После получения реального ID событие удаляется корректно
- ✅ Нет race condition

### Тест 3: Проверка паттернов ID

**В консоли браузера**:
```javascript
// Проверьте временные ID
const tempId1 = 'ev_temp_1732005123456_789';
const tempId2 = 'ev_temp_12345';
const realId = 'e37316';

// ✅ Правильная проверка
console.log(tempId1.startsWith('ev_temp_')); // true
console.log(tempId2.startsWith('ev_temp_')); // true
console.log(realId.startsWith('ev_temp_'));  // false

// ❌ Неправильная проверка (БЕЗ подчеркивания)
console.log(tempId1.startsWith('ev_temp')); // true (работает случайно!)
console.log('ev_tempo_123'.startsWith('ev_temp')); // true (БАГ!)
```
